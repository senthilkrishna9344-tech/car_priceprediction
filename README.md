# 🚗 Car Price Prediction using Machine Learning

## 📌 Project Overview
This project predicts the selling price of used cars using a Linear Regression model. It demonstrates the complete machine learning workflow, including data preprocessing, visualization, model training, and model evaluation.

## 📂 Dataset
The dataset contains information about used cars, including:
- Car Name
- Year
- Present Price
- Selling Price
- Driven Kilometers
- Fuel Type
- Selling Type
- Transmission
- Owner

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## 📊 Project Workflow
1. Import the dataset
2. Data preprocessing
3. Feature engineering (Car Age)
4. Label encoding
5. Correlation analysis
6. Train-test split
7. Linear Regression model training
8. Model evaluation
9. Data visualization

## 📈 Model Performance
- MAE: **1.2147**
- MSE: **3.5370**
- R² Score: **0.8465**

## ▶️ How to Run
1. Clone this repository.
2. Install the required libraries:
   ```
   pip install pandas numpy matplotlib seaborn scikit-learn
   ```
3. Open the Jupyter Notebook.
4. Run all cells.
